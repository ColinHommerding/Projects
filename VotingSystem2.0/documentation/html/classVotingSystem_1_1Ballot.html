Detailed Description

A basic Ballot class.

Contains an ID integer, as well as a vector of Candidate pointers. In IR, the candidates are meant to be ranked by the voter's preference. In OPL, since the rules state each ballot only votes for one candidate, the vector would only contain one entry.
