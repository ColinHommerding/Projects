◆ orderCandidates()

void VotingSystem::OPLParty::orderCandidates ( bool  debug)

Rearranges candidates from those with most votes to least votes.

Parameters
debugan int which toggles randomization of tied candidates
Returns
void

Orders candidates in candidates_ vector in decreasing order by each candidate's numVotes_ attribute. Breaks ties randomly if debug is false. If debug is true, then breaks ties by ordering all tied candidates by order of appearance in the input file, which is equivalent to ordering by ascending ID values.
