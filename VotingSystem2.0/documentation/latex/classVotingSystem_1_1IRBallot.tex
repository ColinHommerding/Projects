\doxysubsection{Detailed Description}
A basic \mbox{\hyperlink{classVotingSystem_1_1IRBallot}{IRBallot}} class that inherits from \mbox{\hyperlink{classVotingSystem_1_1Ballot}{Ballot}}. 

Contains an ID integer, as well as a vector of \mbox{\hyperlink{classVotingSystem_1_1Candidate}{Candidate}} pointers. In IR, the candidates are meant to be ranked by the voter\textquotesingle{}s preference. In OPL, since the rules state each ballot only votes for one candidate, the vector would only contain one entry.
